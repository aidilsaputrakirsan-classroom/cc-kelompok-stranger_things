# 🎓 UTS Demo Script 

## 1. Setup (0–1 menit) — Lead DevOps
- Buka terminal di root project
- Jalankan `docker compose up -d`
- Jalankan `docker compose ps`
- Tunjukkan semua service berjalan

## 2. Frontend Demo (1–3 menit) — Lead Frontend
- Buka http://localhost:3000
- Register user baru
- Login ke aplikasi

## 3. CRUD Demo (3–6 menit) — Lead FE + BE
- Create 2 items
- Edit 1 item
- Search item
- Delete 1 item

## 4. Backend Demo (6–7 menit) — Lead Backend
- Buka http://localhost:8000/docs
- Tunjukkan endpoint documentation

## 5. Docker Persistence Demo (7–8 menit) — Lead DevOps
- Jalankan `docker compose down`
- Jalankan `docker compose up -d`
- Login kembali
- Tunjukkan data masih ada

## 6. Code Walkthrough (8–10 menit) — Lead DevOps
- Tunjukkan `docker-compose.yml`
- Tunjukkan `Dockerfile`
